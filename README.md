# การทำสมองที่สอง (Xambrain)

> ระบบ **สมองที่สอง (externalized cognition)** สำหรับการเรียนรู้และการสอน
> ด้านพฤกษศาสตร์พื้นบ้าน สุขภาพไทย และการใช้พืช
>
> repo/โฟลเดอร์เทคนิค: `botany-xambrain` · ชื่อเผยแพร่: **การทำสมองที่สอง**

---

## 1. โปรเจกต์นี้คืออะไร

**การทำสมองที่สอง** ไม่ใช่แค่เว็บแนะนำอาหารหรือแชทบอท แต่เป็น **แพลตฟอร์มการเรียนรู้**
ที่แยกเป็น 3 ชั้นทำงานร่วมกัน — ครูใช้เก็บความรู้และวัดผล นักเรียนใช้เรียนเองและทดสอบตัวเอง

| ชั้น | ชื่อ | รูปแบบ | สถานะ |
|------|------|--------|--------|
| **1. ความรู้** (Knowledge) | `botany-xambrain` | Obsidian vault / แอปในเครื่อง | ✅ ใช้งานอยู่ (repo นี้) |
| **2. วัดผล** (Assessment) | **ExamFlow** | เว็บ — [test.xambrain.com](https://test.xambrain.com) | ✅ ออนไลน์แล้ว |
| **3. ใช้จริง** (Express) | Food Recommender | เว็บ/แอป (`index.html`) | ✅ prototype ทำงานได้ |

## 2. เป้าหมาย: สมองที่สองสำหรับครูและนักเรียน

### 👩‍🏫 ครู / อาจารย์
- **เก็บความรู้** — concept nodes, สรุปจาก PDF, คลังคำศัพท์
- **ทำคลังข้อสอบ** — qbank 3 ระดับ (L1/L2/L3)
- **วัดผลนักเรียน** — ผ่าน ExamFlow (สร้างข้อสอบ อัปโหลดเฉลย ตรวจอัตโนมัติ สรุปตามตัวชี้วัด)
- **ติวเตอร์** — ดูจุดอ่อนตามตัวชี้วัด แล้วสอนเสริม

### 👨‍🎓 นักเรียน / ผู้เรียนด้วยตนเอง
- **เรียนรู้ด้วยตนเอง** — อ่าน concept ตามลำดับ (Botany Literacy 5 ระดับ)
- **วัดผลตัวเอง** — ฝึกทำข้อสอบบน ExamFlow (ฝึก ไม่ใช่ลอกเฉลย)
- **ประยุกต์ใช้ (L3)** — ใช้เครื่องมือจริง เช่น แนะนำเมนูตามร่างกาย

## 3. กรอบความรู้ (Framework)

### 3 Layer
| Layer | ชื่อ | ครอบคลุม |
|-------|------|----------|
| **S** | พฤกษศาสตร์ (Structure) | สัณฐานวิทยา · การจำแนก · ส่วนของพืช · เนื้อเยื่อ |
| **U** | การใช้ประโยชน์ (Utilization) | อาหาร · ยา · เครื่องใช้ · ย้อมสี · พิธีกรรม |
| **T** | ภูมิปัญญาไทย (Thai Traditional) | สมุนไพร · ตำรับยา · รสยา 9 รส · ธาตุ 4 |

### Botany Literacy 5 ระดับ
`Naming → Identifying → Using → Connecting → Conserving`

### ข้อสอบ 3 ระดับ
- **L1 จำ/ระบุ** → Naming/Identifying
- **L2 เชื่อมโยง** → Using/Connecting
- **L3 วินิจฉัย/ประยุกต์** → Connecting/Conserving (บูรณาการ)

### แกนความรู้ (Foundational axes)
- **AX1** [[food-as-balance]] — อาหารคือสมดุล (4 กระบวนทัศน์: สากล/ไทย/อายุรเวท/จีน)
- **AX2** [[human-unnatural-life]] — มนุษย์ใช้ชีวิตผิดธรรมชาติ (การเคลื่อนไหว + ความสุขจริง vs หลอก)

## 7. หลักการสำคัญ

- ภาษาหลักของเนื้อหาคือ **ไทย** (ยกเว้นศัพท์เทคนิค/ชื่อวิทยาศาสตร์)
- ชื่อวิทยาศาสตร์เขียนแบบ *Genus species* พร้อมวงศ์ (Family)
- ทุกไฟล์มี YAML frontmatter · ใช้ wikilink `[[name]]` · รูป `![[folder/image.png]]`
- **AI เป็นผู้ช่วย ไม่ใช่ผู้ตัดสิน** — กฎ JSON เลือกเมนู, ข้อสอบวัดผล, ครูเป็นเจ้าของความรู้
- **ส่งเสริมสุขภาพเชิงป้องกัน — ไม่ใช่การวินิจฉัยหรือรักษาโรค** (ต้องแสดงคำเตือนในทุกหน้าเผยแพร่)
- **Git** — remote บน GitHub (repo หลัก); ไฟล์ใหญ่ (รูป/PDF) ยังอยู่ local ตาม `.gitignore`
