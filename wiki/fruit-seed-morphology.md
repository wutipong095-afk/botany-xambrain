# ผลและเมล็ด (Fruit & Seed)

> ✅🌐 **source_type: mixed** — ชนิดผลละเอียดจากสไลด์อาจารย์ ([[source-flower-fruit-seed]]) + อ้างอิงเสริมสสวท.

## สาระสำคัญ

หลังการผสมเกสร **รังไข่ (ovary)** เจริญเป็น **ผล (fruit)** และ **ออวุล (ovule)** เจริญเป็น **เมล็ด (seed)** ทำหน้าที่ขยายพันธุ์และกระจายพันธุ์

## จำแนกตามที่มาของรังไข่ (ภาพรวม)

| ชนิด | เกิดจาก | ตัวอย่าง |
|------|---------|----------|
| **ผลเดี่ยว (Simple fruit)** | รังไข่เดียว ในดอกเดียว | ส้ม มะเขือ ฟักทอง แอปเปิ้ล มะม่วง |
| **ผลกลุ่ม (Aggregate fruit)** | หลายรังไข่ ในดอกเดียว | น้อยหน่า สตรอเบอรี |
| **ผลรวม (Multiple fruit)** | รังไข่ของหลายดอกในช่อ | สับปะรด ขนุน สาเก ยอ หม่อน |

## ชนิดผลตามสไลด์อาจารย์ (Simple fruit แบ่งย่อย)

![[assets/ดอก-ผล-เมล็ด/p005-02.jpeg|520]]

### ผลสด (Fleshy fruit)

| ไทย | อังกฤษ | ตัวอย่าง |
|-----|--------|----------|
| ผลเมล็ดแข็ง | drupe | มะม่วง ลำไย มะพร้าว |
| ผลมีเนื้อหลายเมล็ด | berry | มะเขือเทศ องุ่น |
| ผลแบบส้ม | hesperidium | ส้ม มะนาว |
| ผลแบบแตง | pepo/gourd | แตงกวา ฟักทอง |
| ผลแบบแอปเปิ้ล | pome | แอปเปิ้ล ลูกแพร์ |

### ผลแห้ง (Dry fruit)

| ไทย | อังกฤษ | ตัวอย่าง |
|-----|--------|----------|
| ผลแห้งเมล็ดล่อน | achene | ทานตะวัน (ผล) |
| ผลธัญพืช | caryopsis/grain | ข้าว ข้าวโพด |
| ผลปีกเดียว | samara | อโศก |
| ผลเปลือกแข็ง | nut | ถั่วเปลือกแข็ง |
| ผลแตกสองครั้ง | schizocarp | ผักชี |
| ฝักแตกแนวเดียว | follicle | ฝักนม |
| ฝักแบบถั่ว | legume | ถั่ว ฝักถั่ว |
| ฝักแบบผักกาด | silique | กาด คะน้า |

## ส่วนประกอบของผล (ผนังผล / Pericarp)

- **Exocarp** — เปลือกนอก
- **Mesocarp** — ชั้นกลาง (มักเป็นเนื้อในผลสด)
- **Endocarp** — ชั้นในสุด (หุ้มเมล็ด เช่น กะลามะพร้าว)

## ส่วนประกอบของเมล็ด

- **เปลือกหุ้มเมล็ด (Seed coat / testa)** — ป้องกันต้นอ่อน
- **ต้นอ่อน (Embryo)**:
  - **แรดิเคิล (Radicle)** — รากอ่อน
  - **พลูมูล (Plumule)** — ยอดอ่อน
  - **ใบเลี้ยง (Cotyledon)** — ใบเลี้ยงเดี่ยว 1 ใบ / ใบเลี้ยงคู่ 2 ใบ
- **เอนโดสเปิร์ม (Endosperm)** — อาหารสำรอง (เด่นในใบเลี้ยงเดี่ยว เช่น ข้าว)

## เชื่อมกับ Layer U/T

- ผล/เมล็ดเป็น **พืชอาหาร** และ **เครื่องยา**: เมล็ดผักชี ดีปลี พริกไทย มะขามป้อม
