# เนื้อเยื่อพืช (Plant Tissue)

> 🌐 **source_type: external** — ค้นจากอินเทอร์เน็ต (ไม่มีในซีทอาจารย์โดยตรง แต่ซีทลำต้นหน้า 15 อ้างถึง "ท่อลำเลียง" → node นี้ขยายความ) แหล่ง: สสวท. (scimath), วิกิพีเดีย · [[reference-sources]]

## สาระสำคัญ

เนื้อเยื่อพืชคือกลุ่มเซลล์ที่ทำงานร่วมกัน แบ่งเป็น **2 กลุ่มใหญ่**: เนื้อเยื่อเจริญ (ยังแบ่งตัวได้) และเนื้อเยื่อถาวร (เปลี่ยนสภาพทำหน้าที่เฉพาะแล้ว) เป็นระดับที่ **ละเอียดกว่าอวัยวะ** ([[plant-morphology]])

## 1. เนื้อเยื่อเจริญ (Meristematic tissue) — เซลล์เล็ก นิวเคลียสใหญ่ แบ่งตัวได้

| ชนิด | ตำแหน่ง | ผล |
|------|---------|-----|
| **ปลายยอด/ปลายราก** (Apical meristem) | ปลายยอด ปลายราก | การเจริญปฐมภูมิ (ยืดยาว) |
| **ด้านข้าง** (Lateral meristem = cambium) | รอบลำต้น/ราก | การเจริญทุติยภูมิ (ขยายขนาด, เนื้อไม้) |
| **เหนือข้อ** (Intercalary meristem) | เหนือข้อ (หญ้า ไผ่) | ยืดปล้อง |

- **Vascular cambium** แทรกระหว่างไซเล็ม–โฟลเอ็ม → สร้าง secondary xylem/phloem (พบในใบเลี้ยงคู่ทุกชนิด) = ที่มาของ **เนื้อไม้** ([[stem-morphology]] woody stem)

## 2. เนื้อเยื่อถาวร (Permanent tissue) — เปลี่ยนสภาพแล้ว

### 2.1 เชิงเดี่ยว (Simple tissue)
| ชนิด | ผนังเซลล์ | สภาพ | หน้าที่ |
|------|-----------|------|---------|
| **พาเรงคิมา (Parenchyma)** | บาง | มีชีวิต | สะสมอาหาร, สังเคราะห์แสง (chlorenchyma) |
| **คอลเลงคิมา (Collenchyma)** | หนาบางส่วน | มีชีวิต | ค้ำจุนบริเวณที่ยังเติบโต |
| **สเกลอเรงคิมา (Sclerenchyma)** | หนา มี lignin | ตายเมื่อโตเต็มที่ | ค้ำจุนหลัก (fiber, sclereid) |

### 2.2 เชิงซ้อน / ท่อลำเลียง (Complex / Vascular tissue)
- **ไซเล็ม (Xylem)** — ลำเลียง **น้ำ+แร่ธาตุ** จากรากขึ้นบน (tracheid, vessel)
- **โฟลเอ็ม (Phloem)** — ลำเลียง **อาหาร** จากใบไปทั่วต้น (sieve tube, companion cell)

### 2.3 เนื้อเยื่อผิว (Epidermis)
- ผิวนอกสุด มี **คิวทิเคิล** กันน้ำ และ **ปากใบ (stomata)** แลกเปลี่ยนแก๊ส/คายน้ำ

## เชื่อมกับ Layer S (อวัยวะ)

| เนื้อเยื่อ | ปรากฏใน |
|-----------|----------|
| Apical meristem | ตายอด/ตาข้าง → [[stem-morphology]] |
| Vascular cambium → เนื้อไม้ | ลำต้นไม้ยืนต้น → [[stem-morphology]] |
| Xylem/Phloem | ท่อลำเลียงในราก/ลำต้น/เส้นใบ → [[root-morphology]], [[leaf-morphology]] |
| Pericycle (parenchyma) | สร้างรากแขนง → [[root-morphology]] |
| Parenchyma สะสมอาหาร | เหง้า/หัว/รากสะสมอาหาร |

## เชื่อมกับ Layer U/T

- **Parenchyma สะสมอาหาร** ในเหง้า (ขิง ข่า ขมิ้น) = ที่เก็บสารสำคัญของสมุนไพร → [[thai-herbal]]
- **Sclerenchyma/fiber** = เส้นใยพืชสำหรับทำเชือก/สิ่งทอ/หัตถกรรม (Layer U)

## Misconceptions ที่พบบ่อย

- คิดว่า xylem ลำเลียงอาหาร → ที่จริง **xylem=น้ำ, phloem=อาหาร**
- คิดว่าเนื้อเยื่อเจริญมีทั่วต้น → จริงๆ อยู่เฉพาะจุด (ปลายยอด ปลายราก cambium)
