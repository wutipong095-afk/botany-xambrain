# ราก (Root)

## สาระสำคัญ

ราก คืออวัยวะของพืชที่ **ไม่มีข้อ ปล้อง ตา และใบ** มีการเรียงตัวของเนื้อเยื่อสม่ำเสมอตลอดราก โดยทั่วไปเจริญลงดินตามแรงโน้มถ่วง

## หน้าที่ (Function)

1. **ค้ำจุน (Anchorage)** — ยึดพืชให้ทรงตัว
2. **ดูดและลำเลียงน้ำ (Absorption & transportation)** — ดูดน้ำ+ธาตุอาหาร
3. **สร้างฮอร์โมน (Producing hormones)**
4. **หน้าที่พิเศษ** — สะสมอาหาร, หายใจ, สังเคราะห์แสง

## ระบบราก (Root system)

| ระบบ | ลักษณะ | พบใน | ตัวอย่าง |
|------|--------|------|----------|
| **รากแก้ว** (Tap root) | รากหลักจาก radicle เจริญลึก + รากแขนงจำนวนมาก | gymnosperm, ใบเลี้ยงคู่ | ฝรั่ง ขนุน ดาวเรือง |
| **รากฝอย** (Fibrous) | รากเล็กสม่ำเสมอ แผ่กระจายรอบทิศ | ใบเลี้ยงเดี่ยว | ข้าวโพด ตะไคร้ หญ้าคา ขิง ข่า |

## จำแนกตามต้นกำเนิด

1. **Primary root** — จาก radicle โคนใหญ่ เรียวลงปลาย = **รากแก้ว (tap root)**
2. **Secondary root** — เจริญจาก primary root (เนื้อเยื่อ pericycle) ขนานพื้นดิน = **รากแขนง (lateral root)**
3. **Adventitious root** — งอกจากส่วนอื่น (ข้อ ลำต้น กิ่ง ใบ แคลลัส) = **รากพิเศษ** จำแนกเป็น 6 ชนิด: รากฝอย, รากค้ำจุน, รากยึดเกาะ, รากสังเคราะห์แสง, รากหายใจ, รากกาฝาก

## จำแนกตามที่อยู่

- **รากใต้ดิน** (Underground root)
- **รากอากาศ** (Aerial root)

## จำแนกตามหน้าที่พิเศษ

| ชนิด | รายละเอียด |
|------|-----------|
| **1. รากสะสมอาหาร** | จากรากแก้ว: Conical / Napiform / Fusiform · จากรากฝอย: Tuberous / Fascicled |
| **2. Photosynthetic root** | รากสังเคราะห์แสง |
| **3. Supporting root** | ค้ำจุน: Prop / Brace / Buttress |
| **4. Climbing root** | รากปีนป่าย |
| **5. Respiratory root** | รากหายใจ |
| **6. Parasitic root** | รากกาฝาก |

## เชื่อมกับ Layer U/T (การใช้ประโยชน์ + ภูมิปัญญา)

- **รากสะสมอาหาร** หลายชนิดเป็นพืชอาหาร/สมุนไพร เช่น กระชาย
- ต่อยอดสู่ [[stem-morphology]] — เหง้า (ขิง ข่า ขมิ้น) เป็น **ลำต้นใต้ดิน** ไม่ใช่ราก (จุดสับสนที่พบบ่อย!)

## Misconceptions ที่พบบ่อย

- คิดว่า "เหง้าขิง/ข่า/ขมิ้น" เป็นราก → จริงๆ เป็น **ลำต้นใต้ดิน (rhizome)** เพราะมีข้อ ปล้อง ตา
- คิดว่ารากทุกชนิดอยู่ใต้ดิน → มีรากอากาศและรากค้ำจุนเหนือดินด้วย
