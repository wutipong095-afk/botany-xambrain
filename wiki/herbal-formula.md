# ตำรับยา / พิกัดยา (Herbal Formulas)

> 🌐 **source_type: external** — ค้นจากอินเทอร์เน็ต แหล่ง: ม.สงขลานครินทร์, เภสัชกรรมไทย (samunpri), be7herb (คัมภีร์สรรพคุณ) · ดูที่มาท้ายหน้า · [[reference-sources]]

## สาระสำคัญ

**พิกัดยา** คือการนำสมุนไพรหลายชนิดมารวมกันตามอัตราส่วนที่กำหนด เพื่อเสริมฤทธิ์กัน node นี้ยกตัวอย่าง 3 พิกัดคลาสสิก ที่แสดงการเชื่อม **ส่วนพืช (Layer S) → รส (Layer T) → ธาตุ** ได้ชัด

## 1. ตรีกฏุก (Trikatuk) — "ของเผ็ด 3 สิ่ง"

| สมุนไพร | ส่วนพืช (Layer S) | รส |
|---------|---------------------|-----|
| พริกไทย | ผล/เมล็ด → [[fruit-seed-morphology]] | เผ็ดร้อน |
| ขิงแห้ง | เหง้า → [[stem-morphology]] | เผ็ดร้อน |
| ดีปลี | ผล → [[fruit-seed-morphology]] | เผ็ดร้อน |

- **รสประธาน**: ร้อน · **สรรพคุณ**: กระตุ้นธาตุไฟ ช่วยย่อย แก้ลม ดี เสมหะ (นิยมใช้ **ฤดูฝน** / คนธาตุน้ำ)
- โยง [[dhatu-4-plants]] — ยากระตุ้นเมื่อธาตุไฟหย่อน

## 2. ตรีผลา (Triphala) — "3 ผล"

| สมุนไพร | ส่วนพืช (Layer S) | รส |
|---------|---------------------|-----|
| สมอไทย | ผล → [[fruit-seed-morphology]] | ฝาด เปรี้ยว |
| สมอพิเภก | ผล → [[fruit-seed-morphology]] | ฝาด |
| มะขามป้อม | ผล → [[fruit-seed-morphology]] | เปรี้ยว ฝาด |

- อัตราส่วน **1:1:1** (เนื้อในผล) · **สรรพคุณ**: ปรับธาตุ/สมดุล ระบายอ่อนๆ ยาอายุวัฒนะ ต้านอนุมูลอิสระ (มี **แทนนิน + vitamin C**)
- ทั้ง 3 เป็น **ผล** → ตัวอย่างที่ดีของ [[fruit-seed-morphology]] + รสฝาด/เปรี้ยวใน [[herbal-taste-9]]

## 3. เบญจกูล (Benjakul) — "5 ส่วนพืช ประจำ 5 ธาตุ" ⭐

| สมุนไพร | ส่วนพืช (Layer S) | ธาตุที่ดูแล |
|---------|---------------------|--------------|
| ดอกดีปลี | ดอก/ผล → [[flower-morphology]] | ดิน |
| รากช้าพลู | ราก → [[root-morphology]] | น้ำ |
| เถาสะค้าน | ลำต้น(เถา) → [[stem-morphology]] | ลม |
| รากเจตมูลเพลิง | ราก → [[root-morphology]] | ไฟ |
| เหง้าขิงแห้ง | เหง้า → [[stem-morphology]] | อากาศ |

- **รสประธาน**: ร้อน · **สรรพคุณ**: บำรุงธาตุทั้ง 4 ให้บริบูรณ์ ระงับโรคในอาการ 32 (ทวัตติงสาการ)
- เชื่อมตรงกับ [[dhatu-4-plants]] (ตัวยาประจำธาตุ)

## สรุปการเชื่อม 3 Layer

| พิกัด | ส่วนพืชเด่น (S) | รสเด่น (T) | ใช้เมื่อ |
|-------|------------------|-------------|---------|
| ตรีกฏุก | เหง้า+ผล | เผ็ดร้อน | ธาตุไฟหย่อน ย่อยไม่ดี |
| ตรีผลา | ผล 3 ชนิด | ฝาด เปรี้ยว | ปรับธาตุ ระบาย บำรุง |
| เบญจกูล | ครบ 5 ส่วน | เผ็ดร้อน | บำรุง/กระตุ้นธาตุทั้ง 4 |
