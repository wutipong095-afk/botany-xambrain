# อาหารตามธาตุ: ผัก–ผลไม้ รสตามธาตุเจ้าเรือน (Food by Constitution)

> 🌐 **source_type: external** — ค้นจากอินเทอร์เน็ต (ไม่มีในซีทอาจารย์) แหล่งสถาบัน: ศูนย์การแพทย์กาญจนาภิเษก ม.มหิดล · ราชวิทยาลัยจุฬาภรณ์ · สวพส. · รพ.พิษณุโลก · ดูที่มาท้ายหน้า · ระบบอ้างอิง [[reference-sources]]
> 📎 **Curriculum anchor**: หัวข้อนี้ตรงกับรายวิชา **TA301 Food and Nutrition for Applied Thai Traditional Medicine** (ม.ธรรมศาสตร์) — ดู [[reference-sources]] หมวด Curriculum anchors

## สาระสำคัญ

แพทย์แผนไทยถือหลัก **"อาหารเป็นยา"** — ผักและผลไม้ที่คนไทยกินทุกวันมี **รส** ที่ออกฤทธิ์ต่อ **ธาตุทั้ง 4** เช่นเดียวกับสมุนไพร ([[herbal-taste-9]]) การเลือกกินให้ **สมดุลกับธาตุเจ้าเรือน** ของตนเองจึงเป็นการดูแลสุขภาพเชิงป้องกัน node นี้เชื่อม **พืชอาหาร (Layer U)** เข้ากับ **รส–ธาตุ (Layer T)** และย้อนไปยัง **ส่วนพืช (Layer S)**

## ธาตุเจ้าเรือนคืออะไร

**ธาตุเจ้าเรือน (Tard / constitution)** = ธาตุที่เป็นองค์ประกอบใหญ่ที่สุดของร่างกาย เป็นตัวกำหนดลักษณะนิสัย/รูปร่าง/แนวโน้มการเจ็บป่วยของแต่ละคน กำหนดจาก **วัน–เดือน–ปีเกิด** แบ่งเป็น

- **ธาตุเจ้าเรือนตามกำเนิด** — ติดตัวมาแต่เกิด
- **ธาตุเจ้าเรือนปัจจุบัน** — เปลี่ยนได้ตามอายุ ฤดูกาล สิ่งแวดล้อม พฤติกรรม

| ธาตุเจ้าเรือน | เดือนเกิด | ลักษณะเด่น | มักป่วยด้วย |
|--------------|-----------|-----------|-------------|
| **ไฟ (เตโช)** | ม.ค.–มี.ค. | รูปร่างปานกลาง ผิวออกเหลือง ขี้ร้อน หิวบ่อย ใจร้อน | แผลในปาก สิว ผื่น ท้องเสีย |
| **ลม (วาโย)** | เม.ย.–มิ.ย. | รูปร่างผอม ผิวแห้ง ช่างพูด นอนไม่ค่อยหลับ ตกใจง่าย | ท้องอืด ท้องผูก ปวดหัว วิงเวียน |
| **น้ำ (อาโป)** | ก.ค.–ก.ย. | รูปร่างสมบูรณ์ ผิวขาว ตาโต เสียงนุ่ม เชื่องช้า ชอบนอน | อ้วน เสมหะมาก น้ำมูกไหล |
| **ดิน (ปถวี)** | ต.ค.–ธ.ค. | รูปร่างสูงใหญ่ กล้ามเนื้อ/ข้อต่อแข็งแรง | โรคหัวใจ เบาหวาน |

## หลักการเลือกอาหาร: กินรส "ปรับสมดุล" ธาตุ

ธาตุเจ้าเรือนมักเป็นธาตุที่ **มีมาก/กำเริบง่าย** จึงเลือกกินอาหารรสที่ **ปรับ/ทอนธาตุนั้น** ให้สมดุล — หลักเดียวกับ "รสแก้ธาตุพิการ" ใน [[dhatu-4-plants]]

## ตารางใหญ่: ธาตุ → รสควรกิน → ผัก–ผลไม้ ⭐

| ธาตุเจ้าเรือน | รสควรกิน | ผัก–ผลไม้ที่คนไทยกิน (ตัวอย่าง) |
|--------------|----------|-------------------------------|
| **ดิน** | ฝาด หวาน มัน เค็ม | มังคุด ฝรั่งดิบ ฟักทอง เผือก มันเทศ เงาะ กล้วยดิบ ถั่วต่างๆ สมอไทย สะตอ ผักโขม โสน ยอดมะม่วงหิมพานต์ |
| **น้ำ** | เปรี้ยว ขม | มะนาว ส้ม สับปะรด มะเขือเทศ มะยม มะกอก มะระ ดอกแค ใบยอ ใบบัวบก |
| **ลม** | เผ็ดร้อน (สุขุม) | ขิง ข่า ตะไคร้ กระชาย พริกไทย กระเทียม โหระพา ยี่หร่า ช้าพลู กระทือ |
| **ไฟ** | ขม เย็น จืด หวาน | แตงกวา แตงไทย มะระ ตำลึง สายบัว ขี้เหล็ก หัวไชเท้า มะเขือม่วง กะหล่ำปลี เห็ดหูหนู ผักบุ้ง |

> เทียบรสต่อสรรพคุณของแต่ละรสได้ที่ [[herbal-taste-9]] · เทียบรสแก้ธาตุพิการที่ [[dhatu-4-plants]]

## เชื่อม Layer S ⭐ (จุดเด่นของโปรเจกต์)

ผัก–ผลไม้แต่ละชนิดคือ **ส่วนพืชคนละส่วน** — โครงสร้าง (Layer S) จึงบอกได้ตั้งแต่ต้นว่าเรากำลังกิน "อะไรของต้นไม้"

| พืชอาหาร | ส่วนพืชที่กิน (Layer S) | node โครงสร้าง | รส → ธาตุ |
|----------|--------------------------|----------------|-----------|
| ขิง / ข่า / กระชาย | เหง้า (ลำต้นใต้ดิน) | [[stem-morphology]] | เผ็ดร้อน → ลม |
| เผือก / มันเทศ | หัว/รากสะสมอาหาร | [[root-morphology]] · [[stem-morphology]] | มัน หวาน → ดิน |
| ตำลึง / ผักบุ้ง / บัวบก | ใบ/ยอด | [[leaf-morphology]] | จืด เย็น → ไฟ |
| ดอกแค / สายบัว | ดอก/ก้านดอก | [[flower-morphology]] | ขม เย็น → น้ำ/ไฟ |
| มังคุด / ฝรั่ง / มะระ / ฟักทอง | ผล | [[fruit-seed-morphology]] | ฝาด/เปรี้ยว/ขม/มัน |
| ถั่ว / สะตอ | เมล็ด | [[fruit-seed-morphology]] | มัน → ดิน |

## ตัวอย่างวิเคราะห์ครบวง (S → U → T)

**เคส**: คนธาตุลม (เกิด เม.ย.–มิ.ย.) ท้องอืด ผายลมบ่อย
1. ธาตุเจ้าเรือน = **ลม** → ควรกินรส **เผ็ดร้อน** (ตารางบน)
2. เลือกพืชอาหาร: **ขิง** ในต้มข่า/ผัดขิง
3. ส่วนที่กิน (Layer U/S) = **เหง้า** (ลำต้นใต้ดิน [[stem-morphology]])
4. รส **เผ็ดร้อน** ([[herbal-taste-9]]) → ขับลม บำรุงธาตุไฟ → สมดุลธาตุลม ([[dhatu-4-plants]])

**เคส**: คนธาตุไฟ (เกิด ม.ค.–มี.ค.) ขี้ร้อน เป็นสิว แผลในปาก
1. ธาตุเจ้าเรือน = **ไฟ** → ควรกินรส **ขม เย็น จืด**
2. เลือกพืชอาหาร: **มะระ** (แกงจืด/ผัด), **ตำลึง**, **แตงกวา**
3. รส **ขม/จืด/เย็น** → ดับร้อน ลดพิษไข้ → สมดุลธาตุไฟ
