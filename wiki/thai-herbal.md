# สมุนไพรไทย (Thai Herbal Medicine) — hub

> 🌐 **source_type: external** — เนื้อหาเชิงทฤษฎีรสยา/ธาตุ ค้นจากอินเทอร์เน็ต (มีอ้างอิง) · โครงสร้างพืชอ้างอิงซีทอาจารย์ ✅ [[source-plant-morphology]] · ระบบอ้างอิง [[reference-sources]]

## สาระสำคัญ

สมุนไพรไทยคือการนำ **ส่วนของพืช (Layer S)** มาใช้บำบัด โดยพิจารณาจาก **รส (Layer T)** และผลต่อ **ธาตุทั้ง 4** node นี้เป็นสะพานเชื่อมพฤกษศาสตร์กับภูมิปัญญา = แก่นของ botany-xambrain

## หลักการเชื่อม 3 Layer

```
Layer S (ส่วนพืช)  →  Layer U (ส่วนที่ใช้)  →  Layer T (รส + ธาตุ + สรรพคุณ)
  เหง้าขิง          →   เหง้าสด/แห้ง        →   รสเผ็ดร้อน → ขับลม บำรุงธาตุไฟ
  ใบฟ้าทะลายโจร     →   ใบ                  →   รสขม → แก้ไข้ เจริญอาหาร
  ผลมะขาม          →   เนื้อผล             →   รสเปรี้ยว → กัดเสมหะ ระบาย
```

## ส่วนของพืชที่ใช้เป็นยา (เชื่อม Layer S)

| ส่วนพืช | node โครงสร้าง | ตัวอย่างสมุนไพร |
|---------|----------------|------------------|
| ราก / รากสะสมอาหาร | [[root-morphology]] | กระชาย ระย่อม |
| เหง้า / บัลบ์ (ลำต้นใต้ดิน) | [[stem-morphology]] | ขิง ข่า ขมิ้น ไพล กระเทียม |
| ใบ | [[leaf-morphology]] | ฟ้าทะลายโจร ย่านาง กะเพรา |
| ดอก | [[flower-morphology]] | ดอกแค อัญชัน เกสรบัวหลวง |
| ผล / เมล็ด | [[fruit-seed-morphology]] | มะขามป้อม ดีปลี พริกไทย |

## กรอบวิเคราะห์สมุนไพร (Herbal analysis frame)

1. **ระบุพืช** (Layer S: สัณฐานวิทยา ช่วยจำแนก) → [[plant-morphology]]
2. **ส่วนที่ใช้** (Layer U) → ราก/เหง้า/ใบ/ดอก/ผล
3. **รส** (Layer T) → [[herbal-taste-9]]
4. **ผลต่อธาตุ + สรรพคุณ** → เลือกใช้ตามความพิการของธาตุ
