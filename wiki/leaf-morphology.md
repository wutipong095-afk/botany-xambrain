# ใบ (Leaf)

## สาระสำคัญ

ใบเป็นอวัยวะที่เจริญจากข้อของลำต้น ทำหน้าที่หลักคือ **สังเคราะห์แสง (photosynthesis)**, **คายน้ำ (transpiration)** และ **หายใจ (respiration)** ลักษณะของใบเป็นเครื่องมือสำคัญในการ **จำแนกและระบุพืช** (Botany Literacy: Identifying)

## ส่วนประกอบของใบ

- **ก้านใบ (Petiole)** — เชื่อมแผ่นใบกับลำต้น
- **แผ่นใบ (Blade / Lamina)** — ส่วนแบนกว้าง รับแสง
- **เส้นใบ (Venation)** — ท่อลำเลียง; แบบร่างแห (netted — ใบเลี้ยงคู่) vs แบบขนาน (parallel — ใบเลี้ยงเดี่ยว)
- **หูใบ (Stipule)** — (บางชนิด)

## ชนิดของใบ: ใบเดี่ยว vs ใบประกอบ

![[สัณฐานวิทยาของใบ/p001-02.png|500]]

- **ใบเดี่ยว (Simple leaf)** — แผ่นใบเดียวต่อก้าน
- **ใบประกอบ (Compound leaf)** — แบ่งเป็นใบย่อย (leaflet) มี 2 แบบใหญ่:

| แบบนิ้วมือ (Palmately compound) | แบบขนนก (Pinnately compound) |
|-------------------------------|------------------------------|
| มีห้าใบย่อย (pentafoliolate) | มีสามใบย่อย (trifoliolate) |
| มีสามใบย่อย (trifoliolate) | ขนนกปลายคี่ (odd-pinnate) |
| มีสองใบย่อย (bifoliolate) | ขนนกปลายคู่ (even-pinnate) |
| | ขนนกสองชั้น (bipinnate) |
| | ขนนกสามชั้น (tripinnate) |

## การเรียงใบบนลำต้น (Phyllotaxy)

![[สัณฐานวิทยาของใบ/p002-02.png|450]]

1. **สลับ** (alternate)
2. **ตรงข้าม** (opposite)
3. **เป็นวงรอบ** (whorl)
4. **ตรงข้ามสลับตั้งฉาก** (decussate)
5. **เวียน** (spiral)
6. **สลับระนาบเดียว** (distichous)

## รูปร่างใบ (Leaf shape)

![[สัณฐานวิทยาของใบ/p003-02.png|500]]

รูปเข็มแหลม (subulate) · รูปเข็ม (acicular) · รูปแถบ (linear) · รูปขอบขนาน (oblong) · รูปใบหอก (lanceolate) · รูปใบหอกกลับ (oblanceolate) · รูปรี (elliptic) · รูปไข่ (ovate) · รูปไข่กลับ (obovate) · รูปสามเหลี่ยม (deltoid) · รูปสี่เหลี่ยมข้าวหลามตัด (rhomboid) · รูปไต (reniform) · รูปกลม (orbicular) · รูปหัวใจ (cordate) · รูปเคียว (falcate) · รูปช้อน (spathulate)

## ปลายใบ / โคนใบ / ขอบใบ (Apex / Base / Margin)

![[สัณฐานวิทยาของใบ/p004-02.png|500]]

- **ปลายใบ (Apex)**: ปลายคล้ายหาง (caudate), ปลายเรียวแหลม (cuspidate), ปลายมีติ่งหนาม (mucronate), ปลายแหลม (acute), ปลายมน (obtuse), ปลายเว้าตื้น (retuse), ปลายเว้าบุ๋ม (emarginate), ปลายตัด (truncate)
- **โคนใบ (Base)**: รูปลิ่ม (cuneate), สอบเรียว (attenuate), มีติ่งหู (auriculate), มน (rounded), รูปหัวใจ (cordate), รูปเงี่ยงใบ (sagittate), เบี้ยว (oblique), รูปเงี่ยงลูกศร (hastate)
- **ขอบใบ (Margin)**: เรียบ (entire), เป็นคลื่น (undulate), จักฟันเลื่อย (serrate), จักซี่ฟัน (dentate), จักมน (crenate), เว้าลึกแบบขนนก (pinnatifid), เว้าลึกแบบนิ้วมือ (palmatifid)

## เชื่อมกับ Layer U/T (การใช้ประโยชน์ + ภูมิปัญญา)

- ใบหลายชนิดเป็น **ผักและสมุนไพร**: ใบกะเพรา โหระพา ย่านาง ใบเตย มะกรูด → โยงรสยาและธาตุ ([[plant-morphology]] → Layer T)
- ใบใช้ **ย้อมสี** และงาน **หัตถกรรม/ห่ออาหาร** (ใบตอง ใบจาก)
- ลักษณะขอบใบ/รูปใบ/การเรียงใบ = กุญแจสำคัญในการ **ระบุชนิดพืชสมุนไพร** ภาคสนาม
