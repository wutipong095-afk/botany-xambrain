# ใบ (Leaf)

## สาระสำคัญ

ใบเป็นอวัยวะที่เจริญจากข้อของลำต้น ทำหน้าที่หลักคือ **สังเคราะห์แสง (photosynthesis)**, **คายน้ำ (transpiration)** และ **หายใจ (respiration)** ลักษณะของใบเป็นเครื่องมือสำคัญในการ **จำแนกและระบุพืช** (Botany Literacy: Identifying)

## ส่วนประกอบของใบ

- **ก้านใบ (Petiole)** — เชื่อมแผ่นใบกับลำต้น
- **แผ่นใบ (Blade / Lamina)** — ส่วนแบนกว้าง รับแสง
- **เส้นใบ (Venation)** — ท่อลำเลียง; แบบร่างแห (netted — ใบเลี้ยงคู่) vs แบบขนาน (parallel — ใบเลี้ยงเดี่ยว)
- **หูใบ (Stipule)** — (บางชนิด)

## ชนิดของใบ: ใบเดี่ยว vs ใบประกอบ

![[assets/สัณฐานวิทยาของใบ/p001-02.png|500]]

- **ใบเดี่ยว (Simple leaf)** — แผ่นใบเดียวต่อก้าน (เปรียบเทียบกับใบประกอบในภาพ)
- **ใบประกอบ (Compound leaf)** — แบ่งเป็นใบย่อย (leaflet) ต่อก้านใบเดียว; แยก 2 แบบใหญ่:

| # | แบบนิ้วมือ (Palmately compound) | แบบขนนก (Pinnately compound) |
|---|----------------------------------|------------------------------|
| 1 | มีห้าใบย่อย (pentafoliolate) | |
| 2 | มีสามใบย่อย (trifoliolate) — แผ่จากจุดเดียว | |
| 3 | มีสองใบย่อย (bifoliolate) | |
| 4 | | มีสามใบย่อย (trifoliolate) — เรียงตามก้านใบย่อย |
| 5 | | ขนนกปลายคี่ (odd-pinnate) |
| 6 | | ขนนกปลายคู่ (even-pinnate) |
| 7 | | ขนนกสองชั้น (bipinnate) |
| 8 | | ขนนกสามชั้น (tripinnate) |

> หมายเหตุ: trifoliolate #2 (นิ้วมือ) กับ #4 (ขนนก) ต่างกัน — แบบนิ้วมือแผ่จากปลายก้านใบ แบบขนนกเรียงตามก้านใบย่อย

## การเรียงใบบนลำต้น (Phyllotaxy)

![[assets/สัณฐานวิทยาของใบ/p002-02.png|450]]

1. **สลับ** (alternate)
2. **ตรงข้าม** (opposite)
3. **เป็นวงรอบ** (whorl)
4. **ตรงข้ามสลับตั้งฉาก** (decussate)
5. **เวียน** (spiral)
6. **สลับระนาบเดียว** (distichous)

## ประเภทของใบ 6 ประเภท

จาก [[source-leaf-extended]] (อ.ประนอมจันทรโนทัย อ้างอิงในสไลด์):

| # | ประเภท | ลักษณะ |
|---|--------|--------|
| 1 | ใบแท้ (foliolate) | มีคลอโรฟิลล์ สังเคราะห์แสง |
| 2 | ใบดอก (floral leaf) | เปลี่ยนเป็นส่วนดอก สืบพันธุ์ |
| 3 | ใบเลี้ยง (cotyledon) | ในเมล็ด อาจสะสมอาหาร |
| 4 | ใบเกล็ด (scale leaf) | ไม่มีคลอโรฟิลล์ ป้องกันตา/สะสม |
| 5 | ใบประดับ (bract) | รองรับดอก/ช่อ อาจมีสีสวย |
| 6 | ใบดัดแปลง (modified leaf) | หนาม มือเกาะ จับแมลง ฯลฯ |

## การเรียงใบเพิ่มเติม

นอกจาก 6 แบบด้านบน (จาก PDF 4 หน้า):
- **Basal rosette** — เรียงเป็นวงกอที่โคนต้น
- **Equitant** — เรียงซ้อนแบบดาบ (พบในพืชบางวงศ์)
- **Spiral** — (= เวียน ในตาราง phyllotaxy)

## หูใบ (Stipule)

ใบขนาดเล็กที่ข้อใบ (บางชนิด) — ช่วยป้องกันหรือระบุชนิดพืช (ดูรูปในสไลด์ [[source-leaf-extended]] หน้า 29–30)

## ผิวใบและเนื้อใบ

### ผิวใบ (Leaf surface)
- **Glabrous** — เรียบ · **Glaucous** — นวลขาว (ใบตอง ยูคา) · **Pubescent** — มีขนนุ่ม · **Pellucid-punctate** — ต่อมน้ำมัน (มะกรูด)

### เนื้อใบ (Leaf texture)
- **Succulent** — หนาอมน้ำ · **Coriaceous** — หนาเหนียว · **Hyaline** — บางโปร่ง · **Membranous** — บางอ่อน

## ใบดัดแปลง (Modified leaf) — 10 แบบ

| # | ชนิด | ตัวอย่างไทย |
|---|------|-------------|
| 1 | มือเกาะ (tendril) | มะระ แตงกวา ถั่วลันเตา |
| 2 | หนาม (spine) | กระบองเพชร สับปะรด |
| 3 | ใบสะสมอาหาร | ว่านหางจระเข้ หัวหอม กระเทียม |
| 4 | ใบเกล็ด | ขิง ข่า สน |
| 5 | ตาเกล็ด (bud scale) | สาเก |
| 6 | phyllode (ใบก้านสังเคราะห์แสง) | กระถินณรงค์ ปริก |
| 7 | ทุ่นลอย (floating leaf) | ผักตบชวา |
| 8 | ใบประดับ (bract) | เฟื่องฟ้า หน้าวัว |
| 9 | ใบจับแมลง | กาบหอยแครง หม้อข้าวหม้อแกงลิง |
| 10 | ใบสืบพันธุ์ | ต้นตายใบเป็น |

## เชื่อมกับ Layer U/T (การใช้ประโยชน์ + ภูมิปัญญา)

- ใบหลายชนิดเป็น **ผักและสมุนไพร**: ใบกะเพรา โหระพา ย่านาง ใบเตย มะกรูด → โยงรสยาและธาตุ ([[plant-morphology]] → Layer T)
- ใบใช้ **ย้อมสี** และงาน **หัตถกรรม/ห่ออาหาร** (ใบตอง ใบจาก)
- ลักษณะขอบใบ/รูปใบ/การเรียงใบ = กุญแจสำคัญในการ **ระบุชนิดพืชสมุนไพร** ภาคสนาม
