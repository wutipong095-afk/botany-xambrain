# แกงเลียงผักรวม — วิเคราะห์ S/U/T (Tier 3)

> คำแนะนำเชิงส่งเสริมสุขภาพ — ไม่ใช่การวินิจฉัยหรือรักษาโรค

## 1. วัตถุดิบ (Layer S)

| ส่วน | ส่วนพืช | บทบาท |
|------|---------|--------|
| พริกไทย | เมล็ด | เผ็ดร้อน ขับลม-เสมหะ |
| ตำลึง | ใบ/ยอด | ขม เย็น |
| ฟักทอง | ผล | หวาน |
| บวบ | ผล | หวาน เย็น |
| ใบแมงลัก | ใบ | ขม เย็น |

## 2. รส–ฤทธิ์–พลังงาน

| มิติ | สรุป |
|-----|------|
| **A รส** | เผ็ดร้อน (พริกไทย) + หวาน (ฟักทอง/บวบ) + ขม (ตำลึง) |
| **B ฤทธิ์** | **อุ่น** — ขับลม ขับเสมหะ |
| **C พลังงาน** | **ต่ำ** — ผักเป็นหลัก |

## 3. เหมาะกับใคร

- คนธาตุ**ลม/น้ำ**
- หวัด คัดจมูก
- หญิงให้นม (บำรุงน้ำนมตามภูมิปัญญา)
- คุมน้ำหนัก

## 4. ควรระวัง

- แพ้กุ้ง (ถ้าใส่กุ้งแห้ง)
- กระเพาะอ่อน (พริกไทยจัด)

## 5. สรุป

แกงผักสมุนไพร **แคลอรีต่ำ** ครบรส ขับลมและเสมหะ — ต้มสุกล้วน
