# Overview

**botany-xambrain** คือฐานความรู้เกี่ยวกับ **พฤกษศาสตร์พื้นบ้าน (Ethnobotany)** เน้นการเชื่อมโยงพฤกษศาสตร์สมัยใหม่เข้ากับภูมิปัญญาการใช้พืชแบบไทย ผ่านโครงสร้างแบบ knowledge graph

## โฟกัสหลัก

- **พฤกษศาสตร์**: สัณฐานวิทยา, การจำแนกพืช, ส่วนของพืช
- **การใช้ประโยชน์**: อาหาร ยา เครื่องใช้ ย้อมสี พิธีกรรม
- **ภูมิปัญญาไทย**: สมุนไพร ตำรับยา รสยา 9 รส ธาตุ 4
- **เป้าหมาย**: พัฒนา **Botany Literacy** ที่เชื่อมพฤกษศาสตร์สมัยใหม่กับองค์ความรู้ท้องถิ่น

## กรอบ 3 Layer

```
Layer S: พฤกษศาสตร์ (Structure)
  ├── สัณฐานวิทยาพืช (plant morphology)
  ├── ส่วนของพืช: ราก ลำต้น ใบ ดอก ผล เมล็ด
  ├── เนื้อเยื่อพืช (plant tissue)
  └── การจำแนกพืช (taxonomy: วงศ์ สกุล ชนิด)

Layer U: การใช้ประโยชน์ (Utilization)
  ├── พืชอาหาร (food plants)
  ├── พืชสมุนไพร (medicinal plants)
  ├── พืชเครื่องใช้/หัตถกรรม
  ├── พืชย้อมสี (dye plants)
  └── พืชพิธีกรรม/ความเชื่อ

Layer T: ภูมิปัญญาไทย (Thai Traditional)
  ├── สมุนไพรไทย + สรรพคุณ
  ├── ตำรับยา (herbal formula)
  ├── รสยา 9 รส
  ├── ธาตุ 4 (ดิน น้ำ ลม ไฟ)
  └── ความเชื่อและองค์ความรู้ท้องถิ่น
```

## Botany Literacy 5 ระดับ

1. **Naming** — เรียกชื่อพืชได้ (ชื่อพื้นบ้าน + ชื่อวิทยาศาสตร์)
2. **Identifying** — จำแนก/ระบุพืชจากลักษณะสัณฐาน
3. **Using** — รู้วิธีใช้ประโยชน์ (ส่วนไหน ใช้อย่างไร)
4. **Connecting** — เชื่อมโครงสร้างพืช ↔ การใช้ ↔ ภูมิปัญญา
5. **Conserving** — เข้าใจการอนุรักษ์และใช้อย่างยั่งยืน

## ข้อสอบ 3 ระดับ

- **L1 จำ/ระบุ** → map กับ Naming/Identifying
- **L2 เชื่อมโยง** → map กับ Using/Connecting
- **L3 วินิจฉัย/ประยุกต์** → map กับ Connecting/Conserving (บูรณาการ)
